teacher_want.exe:首先，用户名和密码必须输入正确。不然虽然不会报错，也会说发送了多少邮件，但是实际上没有发送一封。
格式，一定按照test.xls来。按照test.xls，
发送的主题是：xls里面的邮件主题这一行。
发件人是xls发件人这一行指定
收件人就是xls收件人指定
发送的内容如下：
jaffer（xls姓名指定） 教授(xls尊称指定)，您好！
您关于成绩信息如下：（相关信息阐述指定）
成绩：56（由后面的指定）
绩点：3
补助：2333
总计：1

注意，邮件主题之前的6项不能改动。必须这个顺序。之后的可以改变添加。改变之后的会加入到邮件发送内容里面。这样可以自己组织后面的内容。

两个dll文件放在C盘system32下面。
然后可以运行。
